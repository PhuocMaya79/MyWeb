---

## 📌 1. Chủ đề

Website portfolio cá nhân.

---

## 📚 2. Nội dung chính

* 👤 **About Me**: Giới thiệu bản thân
* 💡 **Skills**: Các kỹ năng đã học
* 🛠️ **Projects**: Dự án đã thực hiện
* 📞 **Contact**: Thông tin liên hệ

## 📄 3. Cấu trúc website

| Trang       | Mô tả                             |

| 🙋 About    | Thông tin cá nhân                 |
| 📂 Projects | Danh sách dự án                   |
| 📬 Contact  | Liên hệ                           |

  * Banner + ảnh đại diện

* **Projects**

  * Danh sách dự án
